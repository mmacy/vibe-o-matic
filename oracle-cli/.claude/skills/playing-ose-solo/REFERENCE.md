# OSE Solo Reference

## Scaling rules

* **Monsters**: Divide 'Number Appearing' by **7** (round up).
* **Treasure**: Divide total value by **7** (round up).
* **Sidekicks**: If the PC has a retainer, divide by **4** instead.

## Muse themes

Use these valid themes for the `--theme` flag:

| Theme | Focus |
| :--- | :--- |
| **Change** | Plot progression, alteration of state |
| **Social** | NPC interactions, hierarchy, customs |
| **Swords** | Combat, conflict, weapons, strategy |
| **Sorcery** | Magic, curses, artifacts, spells |
| **Divine** | Gods, undead, holy sites, religion |
| **Monstrous** | Beast behavior, appearance, intent |
| **Treasure** | Loot, value, equipment, reward |
| **Wilderness** | Weather, terrain, nature, travel |
| **Talk** | Conversation topics, arguments, rumors |
| **Place** | Architecture, dungeons, settlements |

## Chaos dice pools

Select dice count based on the likelihood of the event triggering:

* **2d6**: Very Likely (Event happens soon)
* **4d6**: Likely (Standard tension)
* **6d6**: Unlikely (Slow burn)
* **8d6**: Very Unlikely
